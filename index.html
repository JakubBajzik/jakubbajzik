Research project - Joint modelling of whole genome sequence data for human height via approximate message passing

  • Github
  • Link

Recent studies suggest the presence of thousands of common genetic variant associations and hundreds of low-frequency or rare variants associated with human height. Here, we show that existing variable selection methods may fail to either select the correct genomic regions, or the appropriate genetic variants due to the complex and potentially long-distance correlation structure of whole genome sequence (WGS) data. To address this, we develop a new algorithmic paradigm based on approximate message passing, gVAMP, which we show accurately localizes associations to variants with the correct frequency and position in the DNA.

Research project - Epigenome-wide association studies using approximate message passing

  • Github
  • Link

Recent technological advances allow for a diverse range of genomic features to be measured within individuals or single cells, that are linked to electronic health information and patient outcomes. However, current state-of-the-art statistical methods for these data do not allow for reliable significance testing and become inefficient in large scales and high dimensions. Here, we propose a new method called gVAMPomi, based on the recently developed genomic Vector Approximate Message Passing (gVAMP) algorithm, which is an iterative algorithm allowing for efficient joint inference of model parameters.

Research project - Genome-wide analysis of multi-ancestry summary statistics using vector approximate message passing

  • Github
  • Link

Combining genome-wide association studies across genetically diverse cohorts of ancestries can potentially leverage the shared information and increase polygenic risk score portability. Current state-of-the-art cross-ancestry summary statistics methods rely on several regularized linear regressions or variational inference schemes. In this work, we first introduce sgVAMP, the novel summary statistics. Secondly, we extend sgVAMP to a multi-cohort setting, which allows for the joint estimation of shared signals across multiple ancestries.

Freelance project - Virtual Trail of the Heroes of the Slovak National Uprising

  • Link

The Trail of the Heroes of the Slovak National Uprising (Cesta hrdinov SNP) is among the most significant hiking trails in Slovakia, as it passes through locations that played a key role in the Slovak National Uprising (SNP). At 770 km in length, the full route is challenging even for experienced hikers and typically requires around 30 days to complete. This project offered an opportunity to hike the SNP trail virtually, part by part, at one's own pace. Participants tracked their activities using smart wearables, with the goal of cumulatively completing the 770 km distance during the autumn period. The exact physical location of the activity was not important. The app displayed progress along the virtual route via a graphical interface, gradually unlocking short articles about historically significant sites related to the SNP, corresponding to the participant’s current virtual position on the trail.

Research project - Independent Channel Residual Convolutional Network for Gunshot Detection

  • Link

Here, we propose a robust approach for dangerous sound event detection (e.g. gunshots) to improve surveillance systems. The most recent works aim to prefer the time-frequency 2-D representation of sound as input to feed convolutional neural networks. We propose Independent Channel Residual Convolutional Network architecture, which analyse three different types of audio representations in the individual channels. 

Research project - Sound event detection system for DCASE 2021 challenge

  • Link

This paper presents the systems proposal for the DCASE 2021 challenge Task 4 (Sound event detection and separation in domestic environments). The aim is to provide the event time localization timestamps in addition to event class probabilities. We propose two systems, based on convolutional neural networks and class activation mapping.
